원격 연결도구(PuTTY, MobaXterm, WinSCP(파일전송용) 설치)
- MobaXterm : https://mobaxterm.mobatek.net/download-home-edition.html
- PuTTY : https://www.chiark.greenend.org.uk/~sgtatham/putty/latest.html
- WinSCP : https://winscp.net/download/WinSCP-6.5.4-Setup.exe/download

enp0s8 내부 IP를 수동으로 설정(enp0s3은 내부에서 사용)
- Wired Settings -> enp0s8 톱니바퀴 -> IPv4 Manual(192.168.56.101, 255.255.255.0, 192.168.56.2, DNS -> 8.8.8.8 입력 후 Apply) -> 재기동

우클릭 오픈 터미널
저장소 업데이트 
$sudo apt update

net-tools 설치(ifconfig용)
$sudo apt install net-tools vim openssh-server

wait 발생 : waiting for cache lock could not get lock /var/lib/dpkg/lock-frontend. it is held by
- lock 걸린 캐시파일 삭제 후 재 업데이트
$sudo rm /var/lib/apt/lists/lock
$sudo rm /var/cache/apt/archives/lock
$sudo rm /var/lib/dpkg/lock*

$sudo dpkg --configure -a
$sudo apt update

ping test로 정상적을 되는지 확인
ping -c 2 192.168.56.1 - 로컬 컴퓨터
ping -c 2 8.8.8.8 - 구글(외부망)

핑이 안되면 방화벽 해제 후 테스트 필요
